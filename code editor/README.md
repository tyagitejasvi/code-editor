# ProCode Editor

A modern, responsive code editor for writing and testing HTML, CSS & JavaScript code in real-time.

## 🚀 Live Demo

🔗 **[Open Live Demo](https://tyagitejasvi.github.io/code-editor)**

## 🔧 Clone this repository

```bash
git clone https://github.com/tyagitejasvi/code-editor.git
```

## 🛠️ Install & Run

1. Open `codeeditor.html` in your browser (or use local server)
2. Start coding in HTML/CSS/JS panels
3. See live output instantly

## 🎯 Features

- Live Preview
- Auto-save using localStorage
- Undo (Ctrl+Z)
- Download as HTML
- Keyboard shortcuts
